My Journey as a Frontend Developer


How It All Began


My name is Nawaraj Gajurel also know as Nux Gajurel , and I have been passionate about technology since childhood. I always wondered how websites worked, how buttons responded to clicks, and how animations made web pages look so dynamic.My curiosity led me to explore the world of web development, and that’s when I realized that I wanted to become a frontend developer.

Where My Interest in Coding Started


My interest in coding started when I was introduced to basic programming concepts. At first, I was fascinated by how simple lines of code could perform actions. As I explored further, I learned about HTML, CSS, and JavaScript, which made me realize how websites are built. Creating my first basic webpage was a thrilling experience, and from that moment, I knew I wanted to dive deeper into frontend development.

First Steps in Web Development


I started learning HTML and CSS to understand how web pages are structured and styled. It was exciting to see how small changes in code could completely transform a webpage’s appearance. As I progressed, I discovered CSS frameworks like Tailwind CSS, which made designing websites more efficient and visually appealing.

Learning JavaScript & Interactive Web Development


Once I was comfortable with HTML and CSS, I realized that websites needed interactivity, and that’s where JavaScript came in. Learning DOM manipulation, event handling, and ES6 features opened up new possibilities. JavaScript made my projects come to life, and I started building interactive components like sliders, modals, and forms.

Discovering Frontend Frameworks


As I gained confidence in JavaScript, I explored React.js, a popular frontend library that simplifies web development. Learning about components, state management, and hooks was a challenging yet rewarding experience. React helped me create dynamic and scalable web applications, making me feel like a true developer.

Building and Deploying Projects


One of the best parts of my journey has been working on real projects. From simple landing pages to full-fledged web applications, each project has been a step forward in my learning process. Deploying my websites using GitHub Pages, Vercel, and Netlify gave me hands-on experience with version control and hosting.

Challenges and Future Goals


Like every developer, I faced many challenges—debugging errors, struggling with complex concepts, and dealing with performance issues. However, persistence, practice, and the support of the developer community helped me overcome these hurdles. Looking ahead, I aim to learn backend development, APIs, and full-stack technologies to expand my skill set.

Conclusion


My journey as a frontend developer has been an incredible learning experience. From writing my first HTML tag to building dynamic web applications, every step has been a new adventure. Through this blog series, I hope to document my progress, share my knowledge, and inspire others who are starting their own journey in web development.
